# HTML Snippets

Поместите HTML-фрагменты в `src/content/services/...` и они автоматически попадут в бандл через `import.meta.glob`.

- Единичный блок — создайте файл `services/<service>/<name>.html` и используйте `html('<service>/<name>')` в `services.js`.
- Набор блоков — создайте папку `services/<service>/<group>/` и добавьте в неё файлы `01.html`, `02.html`, ...; восстановить их можно вызовом `htmlList('<service>/<group>')`, элементы будут отсортированы по имени файла.
- Можно добавлять теги `<p>`, `<ul>`, `<strong>` и любые inline-элементы без дополнительного экранирования — компоненты уже используют `dangerouslySetInnerHTML`.

Файлы обнаруживаются автоматически, перезапуск dev-сервера не требуется.
